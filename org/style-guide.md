# Style Guide

## Main Colors

- **Background**: `#191923`
- **Accent**: `#22CC99`
- **Hover**: `rgb(21, 184, 135)`

### Buttons

```
button {
    border: .1em solid #22CC99;
    border-radius: .3em;
    font-size: inherit;
    padding: .5em;
    margin: .5em;
    outline: none;
    background-color: #22CC99;
    cursor: pointer;
}

button:focus {
    border: .1em solid lightgrey;
}

button:hover {
    background-color: rgb(31, 199, 149);
    border-color: rgb(31, 199, 149);
}
```
